DEVOIR DE CONTROLE CONTINU - 2022/2023
Projet réalisé dans le cadre de le l'UE "Méthode de conception" de la troisième année de licence informatique à l'université de Caen


******************************************************

L’objectif de ce projet est de réaliser des développements autour de jeux de cartes, menant en particulier à la création d’un jeu de « blackjack ».

Scripts pour lancer le programme :

Le programme blackjack peut etre lancer grâce à l'IDE Eclipse en créant un nouveau projet à partir de Carte et Blackjack. 
Il est également possible de décommenter "Blackjack.jouer();" dans la classe test pour jouer à partir de la console.

Egalement taper la commande "ant" dans un terminal à l'intérieur de la racine du projet ou de l'éxecuter dans un IDE permet de re-générer le contenu de dist ainsi que la compilation de l'application.
Néanmoins nous avons eu des problèmes avec le fichier ant et le fait d'utiliser javaFX, c'est pour cela qu'il est nécessaire de lancer l'application avec un IDE et la librairie JavaFX 19 dans le dossier lib de blackjack.

Il faut également implémenter une librairie JUNIT afin que les tests marchent.
